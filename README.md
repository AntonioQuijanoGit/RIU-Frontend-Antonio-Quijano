# RIU Frontend - Superhero Management

Angular 19 SPA for superhero CRUD operations with Material Design, dual view modes (cards/table), real-time search, pagination, and 81% test coverage.

## Features
- Create, read, update and delete superheroes
- Dual view modes: cards and table
- Real-time search and filtering by publisher
- Pagination with customizable items per page
- Form validation and delete confirmations

## Tech Stack
Angular 19 • Angular Material • RxJS • TypeScript • Docker • Nginx

## Quick Start

**Local:**
```bash
npm install
ng serve
# http://localhost:4200
```

**Docker:**
```bash
docker build -t riu-frontend .
docker run -p 8080:80 riu-frontend
# http://localhost:8080
```

## Testing
```bash
npm test  # 76 unit tests, 81% coverage
```
